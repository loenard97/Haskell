# Real World Haskell

Coding examples and tutorials while going through the book `Real World Haskell`.

Compile with stack:
```sh
stack ghc src/<file>.hs
```

Compiler flags:
Warn about incomplete pattern matching:
```sh
ghc -fwarn-incomplete
```

Warn shadowed variables:
```sh
ghc -fwarn-name-shadowing
```
